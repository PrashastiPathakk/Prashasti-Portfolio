Building robust, scalable, and user-centric software solutions. This portfolio showcases my journey as a Software Developer and SDE Intern, focusing on full-stack development, efficient algorithms, and impactful systems.

## 🚀 Live Demo
[Prashasti-Portfolio](https://github.com/PrashastiPathakk/Prashasti-Portfolio)

### 🏗️ Engineering Focus
Built with a focus on impact and scalability:
- **devTinder**: A networking platform for developers with efficient matching logic and discovery engines.
- **HMS (Hospital Management System)**: Streamlining resource allocation and patient record management.
- **DSA Tracker**: Performance visualization for complex problem-solving patterns.
- **Scalable State**: RESTful APIs designed for high concurrency and resilient data flow using MongoDB.

### 🧪 System "Under the Hood"
The portfolio includes an interactive **Architecture Blueprint** (built with Mermaid.js) that details the full system flow from the Next.js frontend to the Node.js/Express backend and Persistence Layers.

## 🛠️ Tech Stack
- **Frontend**: Next.js 15 (App Router), React 19, Framer Motion
- **Styling**: Tailwind CSS 4 (Glassmorphism & Custom Tokens)
- **Backend Architecture**: Node.js, Express.js, MongoDB (Mongoose)
- **Security**: JWT, Auth Middlewares, Rate Limiting
- **DevOps**: GitHub Actions, Git

## 📂 Features
- **Dynamic GitHub Stats**: Real-time integration with GitHub API for live commit and repo tracking.
- **Software Principles Section**: Showcasing adherence to REST API Design, MVC, and Clean Architecture.
- **Blog (Engineering Thoughts)**: Deep-dives into technical challenges (e.g., Scaling AI Scanners).
- **Responsive Design**: Fluid experience across mobile, tablet, and ultra-wide desktops.
- **Dark/Light Mode**: Seamless theme switching with persistent preferences.

## 📥 Setup Instructions

### Prerequisites
- Node.js 18+ 
- npm or pnpm

### Local Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/PrashastiPathakk/Prashasti-Portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
